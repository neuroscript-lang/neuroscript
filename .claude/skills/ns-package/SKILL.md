## Workflows

Given: `$ARGUMENTS` — the package management task to perform.

### Initialize a Package

```bash
neuroscript init <name> [--author "Name <email>"] [--license MIT] [--version 0.1.0] [--bin]
```

Creates: `Axon.toml`, `README.md`, `.gitignore`, `src/` directory.
With `--bin`: also creates `examples/` directory.

**Package naming rules**:
- Lowercase alphanumeric + hyphens only
- No leading/trailing/consecutive hyphens
- No underscores

### Add Dependencies

```bash
# Version dependency (for future registry)
neuroscript add core-primitives --version "^1.0"

# Git dependency
neuroscript add attention-blocks --git "https://github.com/org/attention.git" --branch main

# Local path dependency (for development)
neuroscript add local-dev --path ../local-dev
```

Cannot specify both `git` and `path` for the same dependency.

### Fetch Dependencies

```bash
neuroscript fetch           # fetch all deps, generate Axon.lock
neuroscript fetch --verbose # with detailed output
neuroscript fetch --update  # update to latest compatible versions
```

Fetched deps cached at `~/.neuroscript/`. Lockfile (`Axon.lock`) pins exact versions for reproducibility.

### Generate Signing Keys

```bash
neuroscript keygen <name>
```

Creates Ed25519 keypair:
- Private: `~/.neuroscript/keys/<name>.key` (0600 permissions)
- Public: `~/.neuroscript/keys/<name>.pub`

### Publish (Sign & Checksum)

```bash
neuroscript publish                     # sign with auto-discovered key
neuroscript publish --key /path/to.key  # explicit key
neuroscript publish --no-sign           # checksums only, no signature
neuroscript publish --verbose
```

Steps:
1. Computes SHA-256 checksums for all `.ns` files
2. Computes overall checksum from sorted file checksums
3. Signs with Ed25519 (unless `--no-sign`)
4. Updates `[security]` section in Axon.toml

### Verify Package

```bash
neuroscript verify           # verify checksums and signature
neuroscript verify --verbose
```

Checks:
- Per-file checksums match
- Overall checksum matches
- Signature is valid (if present)

### Package Structure

```
my-package/
├── Axon.toml              # package manifest
├── Axon.lock              # dependency lockfile (generated)
├── README.md
├── .gitignore
└── src/
    ├── neuron1.ns
    └── neuron2.ns
```

## Axon.toml Format

See `references/axon-toml-spec.md` for complete format.

Quick reference:
```toml
[package]
name = "my-neurons"
version = "0.1.0"
authors = ["Name <email>"]
license = "MIT"
description = "Description"

neurons = ["NeuronA", "NeuronB"]

[dependencies]
core-primitives = "1.2.0"
attention = { git = "https://github.com/org/attention", branch = "main" }
local-dev = { path = "../local-dev" }

[python-runtime]
requires = ["torch>=2.0"]

[security]
# Generated by neuroscript publish — do not edit manually
```
