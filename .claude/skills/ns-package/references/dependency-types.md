# Dependency Types

## Simple Version

For future central registry:

```toml
[dependencies]
core-primitives = "1.2.0"
```

Supports semver version constraints:
- `"1.2.0"` — exact version
- `"^1.2"` — compatible with 1.2.x (Cargo default)
- `">=1.0, <2.0"` — range

**Note**: Central registry is not yet implemented (Phase 4). Version-only deps are skipped during fetch with a warning.

## Git Dependencies

Fetch from git repositories:

```toml
[dependencies]
# Latest default branch
my-dep = { git = "https://github.com/org/repo" }

# Specific branch
my-dep = { git = "https://github.com/org/repo", branch = "main" }

# Tagged release
my-dep = { git = "https://github.com/org/repo", tag = "v1.0.0" }

# Pinned commit
my-dep = { git = "https://github.com/org/repo", rev = "abc123def" }
```

- Cloned to `~/.neuroscript/git/<hash>/`
- The hash is derived from the git URL for cache deduplication
- If the dependency has a `[security]` section, checksums are verified on fetch

**Precedence**: `rev` > `tag` > `branch` > default branch

## Path Dependencies

For local development:

```toml
[dependencies]
local-lib = { path = "../local-lib" }
```

- Resolved relative to the Axon.toml location
- No caching — uses the path directly
- Ideal for monorepo or multi-package development

## Detailed Dependency Fields

| Field | Type | Description |
|-------|------|-------------|
| `version` | String | Semver constraint |
| `git` | String | Git repository URL |
| `branch` | String | Git branch name |
| `tag` | String | Git tag name |
| `rev` | String | Git commit hash |
| `path` | String | Local filesystem path |
| `optional` | Boolean | Only include if explicitly requested |

## Constraints

- Cannot specify both `git` and `path` for the same dependency
- `branch`, `tag`, `rev` only valid with `git`
- Dependency names follow same rules as package names

## Lockfile (Axon.lock)

Generated by `neuroscript fetch`. Pins exact versions:

```toml
# @generated by neuroscript — do not edit
version = 1

[[package]]
name = "attention-blocks"
version = "0.3.1"
source = "git+https://github.com/org/attention?rev=abc123"

[[package]]
name = "local-dev"
version = "0.1.0"
source = "path+/Users/you/projects/local-dev"
```

Commit `Axon.lock` to version control for reproducible builds.

## Cache Structure

```
~/.neuroscript/
├── git/           # cloned git dependencies
│   ├── <hash1>/   # one dir per unique git URL
│   └── <hash2>/
└── keys/          # signing keypairs
    ├── my-pkg.key
    └── my-pkg.pub
```
