# Axon.toml Specification

## [package] Section

Required metadata for the package.

| Field | Required | Type | Description |
|-------|----------|------|-------------|
| `name` | Yes | String | Package name (lowercase, alphanumeric + hyphens) |
| `version` | Yes | String | Semver version (e.g., "0.1.0", "1.2.3-alpha") |
| `authors` | No | Array[String] | Author list: `"Name <email>"` format |
| `license` | No | String | SPDX license identifier (e.g., "MIT", "Apache-2.0") |
| `description` | No | String | Brief package description |
| `repository` | No | String | Source repository URL |
| `homepage` | No | String | Project homepage URL |
| `documentation` | No | String | Documentation URL |
| `keywords` | No | Array[String] | Search keywords |
| `categories` | No | Array[String] | Category tags |

### Name Validation

- Lowercase letters, digits, and hyphens only
- Must start with a letter
- No trailing hyphens
- No consecutive hyphens (`--`)
- No underscores

Valid: `core-primitives`, `attention123`, `my-cool-neurons`
Invalid: `Invalid_Name`, `-starts-with`, `double--hyphen`

### Version Validation

Must be valid semver: `MAJOR.MINOR.PATCH[-prerelease][+build]`

## neurons Array

Lists neuron names this package provides:

```toml
neurons = ["MultiHeadAttention", "ScaledDotProductAttention", "CrossAttention"]
```

## [dependencies] Section

```toml
[dependencies]
# Simple version constraint
core-primitives = "1.2.0"

# Detailed with git source
attention = { version = "0.3", git = "https://github.com/org/attention" }

# Git with branch/tag/rev
utils = { git = "https://github.com/org/utils", branch = "main" }
stable = { git = "https://github.com/org/stable", tag = "v1.0.0" }
pinned = { git = "https://github.com/org/pinned", rev = "abc123" }

# Local path (development)
local-dev = { path = "../local-dev" }

# Optional dependency
extra = { version = "1.0", optional = true }
```

**Constraint**: Cannot specify both `git` and `path` for the same dependency.

## [python-runtime] Section

Python package requirements for `impl:` references:

```toml
[python-runtime]
requires = ["torch>=2.0", "einops>=0.6"]
```

## [security] Section

Generated by `neuroscript publish` — do not edit manually.

```toml
[security]
publisher-key = "ED25519:29da85daa608..."
signature = "ED25519:a1b2c3d4e5f6..."
checksum = "sha256:93cd5af901067..."

[security.checksums]
"src/attention.ns" = "sha256:4128bddc..."
"src/projection.ns" = "sha256:7f3a2b1c..."
```

| Field | Format | Description |
|-------|--------|-------------|
| `publisher-key` | `ED25519:<64-hex>` | 32-byte Ed25519 public key |
| `signature` | `ED25519:<128-hex>` | 64-byte Ed25519 signature of overall checksum |
| `checksum` | `sha256:<64-hex>` | Overall SHA-256 of sorted file checksums |
| `checksums` | Map[path → hash] | Per-file SHA-256 checksums |

## Complete Example

```toml
[package]
name = "transformer-toolkit"
version = "1.0.0"
authors = ["ML Team <ml@example.com>"]
license = "Apache-2.0"
description = "Production transformer components for NeuroScript"
repository = "https://github.com/org/transformer-toolkit"
keywords = ["transformer", "attention", "nlp"]
categories = ["deep-learning", "nlp"]

neurons = [
    "EncoderBlock",
    "DecoderBlock",
    "CrossAttention",
    "RotaryAttention"
]

[dependencies]
core-primitives = "^2.0"
positional-encoding = { git = "https://github.com/org/pos-enc", tag = "v1.0.0" }

[python-runtime]
requires = ["torch>=2.0", "flash-attn>=2.0"]
```
